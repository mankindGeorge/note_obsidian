# 简历使用说明

本文件夹包含针对三个岗位方向优化的简历版本。

## 文件结构

### Markdown版本（内容源）
- [[通用信息]] — 个人信息、教育经历、荣誉奖励（备份参考）
- [[数据分析]] — 数据分析岗位版本
- [[Web前后端]] — Web前后端岗位版本
- [[AI行业]] — AI行业岗位版本

### HTML版本（可投递）
- [[数据分析岗位简历.html]] — 数据分析岗位简历（浏览器打开可预览/打印）
- [[Web前后端岗位简历.html]] — Web前后端岗位简历
- [[AI行业岗位简历.html]] — AI行业岗位简历
- [[简历样式.css]] — 简历样式表

## 使用方法

1. 双击打开对应的 `.html` 文件
2. 按 `Ctrl+P` 打印
3. 选择"另存为PDF"即可生成可投递的简历文件
