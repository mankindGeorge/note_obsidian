1. **确定研究问题**（来自经济金融或实际场景）→ 明确因变量 `Y`

8. **模型应用**：预测、影响因素分析、控制

# 第二章 一元线性回归模型

## 核心考点

### 1. 最小二乘估计法（OLS，必考）

- **基本原理**：使残差平方和最小
- **必须背诵公式**：`β̂₀`、`β̂₁` 的计算公式
- **优良性质**：无偏性、有效性、一致性（三个性质同时满足，因此是经典估计方法）

### 2. 三大检验（必考，选择题/简答题高频）

| 检验方法 | 解决的核心问题 | 核心逻辑 | 关键公式 |
| --- | --- | --- | --- |
| `F` 检验 | 模型整体的存在性 | 原假设：所有 `X` 对 `Y` 的影响都不显著；通过则说明至少有一个 `X` 对 `Y` 影响显著 | `F = (ESS/k) / (RSS/(n-k-1))` |
| `t` 检验 | 单个自变量的显著性 | 原假设：单个自变量的系数为 0；逐个检验每个 `X` 的贡献 | `t = β̂ᵢ / se(β̂ᵢ)` |
| 相关系数检验 | 变量间线性相关程度 | 检验样本相关系数是否显著不为 0 | `r = Σ(xᵢ-x̄)(yᵢ-ȳ) / √(Σ(xᵢ-x̄)²Σ(yᵢ-ȳ)²)` |

**公式说明**：
- `ESS`：回归平方和（解释变差）；`RSS`：残差平方和（未解释变差）
- `k`：自变量个数；`n`：样本量
- `se(β̂ᵢ)`：回归系数的标准误差

### 3. 残差分析（必考选择题）

- 老师比喻：**残差就像人的血液**，通过残差可以直接反映模型的好坏
- 合格残差标准：零均值、同方差、服从正态分布 `N(0, σ²)`
- **分析目的**：验证模型是否满足基本假定

### 4. 预测（必考）

- 分类：点预测、区间预测
- **易错点**：单点预测和均值预测的点预测值完全相同，但区间预测结果不同
- **区间宽度**：均值预测区间 < 单点预测区间

# 第三章 多元线性回归模型

## 核心考点

### 1. 多元线性回归的 5 个基本假定（必考简答题）

老师强调：**这是最小二乘估计法的使用前提，不满足则 OLS 效果极差或完全失效**。

1. 残差服从正态分布 `N(0, σ²)`（零均值、同方差）
2. 样本量 `n` > 自变量个数 `p+1`（数据行数 > 列数）
3. 自变量之间不存在完全线性相关性（保证 `X^TX` 可逆）
4. 残差之间无自相关
5. 自变量是非随机变量

### 2. 参数估计（必考矩阵公式）

- **矩阵形式**：`β̂ = (X^TX)^{-1}X^TY`
- **前提**：`X^TX` 必须可逆 → 直接引出多重共线性问题
- 满足假定下，OLS 仍具有无偏性、有效性、一致性

### 3. 中心化与标准化（必考目的）

- **唯一目的**：消除自变量量纲影响，使系数具有可比性，从而对自变量的影响程度排序
- **方法**：先中心化（变量减均值），再标准化（除以标准差）
- **应用**：标准化后，系数绝对值越大，该自变量对 `Y` 的影响越大，便于抓住主要矛盾

# 第四章 模型诊断：违背基本假定（一）+ 特殊点诊断

## 一、异方差（三大违背假定之一，必考）

1. **概念**：残差的方差不是恒定值，会随着 `X` 的变化而变化
2. **直观诊断**：绘制残差-X 散点图
   - 同方差：残差在 0 上下均匀波动，幅度恒定
   - 异方差：残差波动幅度随 `X` 变化（先小后大/先大后小/不规则）
3. **统计检验**：等级相关系数检验法（最客观，避免主观看图判断）
4. **修正方法**：加权最小二乘法（WLS）
   - **原理**：在残差平方和前乘以权重，压缩大残差的影响
   - **权重原则**：与残差大小成反比 → 残差越大，权重越小；残差越小，权重越大（最大为 1）
5. **延伸考点**：异方差的产生原因、带来的危害（简答题）

## 二、自相关（三大违背假定之二，必考）

1. **概念**：不同观测值的残差之间存在相关性
2. **诊断方法**：`DW` 检验法
3. **修正方法**：迭代法、差分法
4. **延伸考点**：自相关的产生原因、带来的危害（简答题）

## 三、通用修正方法

- **Box-Cox 变换**：可同时解决异方差和自相关问题，但效果一般（"什么都能做但什么都不精通"）

## 四、特殊点诊断（必考概念与区别，选择题高频）

老师强调：**必须严格区分三个点的定义、诊断标准和因果关系**

1. **异常点**：从 `Y` 轴角度判断，学生化残差 > 3（或删除学生化残差 > 1.3）
2. **高杠杆点**：从 `X` 轴角度判断，杠杆值 > 2 倍平均杠杆值
   - 分类：好的高杠杆点（对模型无影响）、坏的高杠杆点（对模型有显著影响）
3. **强影响点**：对参数估计和假设检验有破坏性影响的点（是结果，不是原因）
   - 产生原因：可能是异常点、坏的高杠杆点，或两者共同导致
   - **诊断方法**：`Cook` 距离，`Cook` 距离 > 1 则判定为强影响点

# 第五章 变量选择与逐步回归（必考）

## 核心原则

**变量选择的基本原则**：少而精 → 自变量个数尽可能少，且每个都显著

## 两个核心方法

1. **所有子集法**
   - **原理**：若有 `M` 个自变量，共产生 `2^M-1` 个回归模型，从中选择最优
   - **缺点**：当 `M` 较大时，模型数量呈指数增长，计算量极大

2. **逐步回归法（必考操作原理）**
   - **原理**：有进有退，每一步同时考虑引入新的显著变量和剔除已引入的不显著变量
   - 老师承诺：**能完整写出逐步回归的操作流程即可得满分**

## 四个选择准则（必考）

| 准则 | 判断标准 |
| --- | --- |
| 调整的 `R²` | 越大越好 |
| `AIC` 准则 | 越小越好 |
| `BIC` 准则 | 越小越好 |
| `CP` 准则 | 越小越好 |

# 第六章 模型诊断：违背基本假定（三）—— 多重共线性（必考）

## 核心考点

1. **概念**：自变量之间存在线性相关性
2. **本质特征**：变量多 + 变量间相关
3. **诊断方法（必考对号入座，会混考选择题）**
   - 方差膨胀因子法（`VIF`）
   - 特征根法
   - 条件数法
   - 直观判断法（约 6-7 种，上课讲过）
4. **优化方法（必考，注意用词区别）**
   - **删除变量**：直接消除多重共线性关系（从根源解决）
   - **增大样本量**：万能方法，有条件可优先使用
   - **有偏估计**：减弱多重共线性对建模的影响（不能消除关系本身）
     - 包括：岭回归、主成分回归

# 第七章 有偏估计方法

## 一、岭回归（必考）

1. **产生背景**：多重共线性导致 `X^TX` 不可逆，OLS 估计完全失效
2. **核心思想**：在 `X^TX` 矩阵对角线上加一个小的正数 `k`（岭参数），强制保证矩阵可逆
3. **岭参数 `k` 的选择**：岭迹法
4. **岭回归的变量选择准则**
   - 系数绝对值很小（< 0.1）的变量删除
   - 岭迹图中系数一直不稳定的变量删除
   - 既小又不稳定的变量优先删除

## 二、主成分回归（老师推荐使用）

### 为什么是多重共线性的克星

| 多重共线性特点 | 主成分特点 |
| --- | --- |
| 变量多 | 变量少（降维） |
| 变量间存在相关性 | 变量间完全无相关性 |

### 核心操作

- **主成分个数选择**：累计方差贡献率 ≥ 80%
- **关键概念**：
  - **载荷**：主成分线性组合中原始变量的系数
  - **得分**：主成分的具体取值（作为新自变量代入回归）
- **注意事项**：降维必然丢失信息，需控制信息丢失量 ≤ 20%

# 第八章 非线性回归模型

## 一、可线性化的非线性回归

1. **可线性化条件**：除 `X` 或 `Y` 以外，模型中不含其他未知参数
2. **常见形式**：
   - **多项式回归**：如 `X²` 项（注意：`X₁` 和 `X₁²` 是两个完全不同的变量）
   - **交互项回归**：`X₁X₂`（表示变量间的"化学反应"，即一个变量的影响依赖于另一个变量的取值）

## 二、不可线性化的非线性回归

1. **参数估计方法**：高斯-牛顿迭代法（不再使用最小二乘法）
2. **核心特点**
   - 需要先给定参数初始值，通过迭代收敛得到最终估计值
   - 平方和分解式 `SST=SSR+SSE` 不再成立
   - 模型形式通常是给定的（如药物反应模型），我们只做参数估计，假设检验被弱化

# 第九章 含定性变量的回归模型

## 一、自变量 `X` 是定性变量

1. **虚拟变量个数确定（必考易错点）**：严格遵循 `k-1` 原则，`k` 为定性变量的分类数
   - 例：性别（2 分类）→ 1 个虚拟变量；季节（4 分类）→ 3 个虚拟变量
   - **绝对错误**：一个定性变量对应一个虚拟变量
2. **量化方式**：标准 0-1 编码
3. **系数解释**：与连续变量不同，表示不同类别之间 `Y` 的平均差异

## 二、因变量 `Y` 是定性变量（必考二分类情况）

1. **传统 OLS 的三大破坏**
   - 残差非正态（服从两点分布，离散型）
   - 存在异方差
   - 预测值可能超出 `[0,1]` 范围（左边不等于右边）
2. **标准解决方案**：Logistic 回归模型
   - **变换形式**：`ln(P/(1-P)) = β₀ + β₁X`
   - 多分类情况：在二分类基础上扩展
   - 定序定性变量（如学历：小学/初中/高中）：仅作了解，不考
