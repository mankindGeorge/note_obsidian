# 项目规则

## Markdown
### 标题
- 使用 # / ## / ###
- 标题之间不空行
- 标题与正文空一行

### 代码块
- 前后空一行
- 上方写说明

### 列表
- 有序：步骤
- 无序：要点
- 不空行
- Tab 缩进

### 强调
- 行内代码：`参数 / 文件 / 路径 / 代码`
- 加粗：重点 / 说明 / 定义

---

## 结构
- 使用 --- 分隔章节
- 代码前写说明
- 代码内中文注释

---

## 标点
- 中英文之间加空格
- 中文标点直接用，不加空格
- 英文符号前后加空格

---

## 文件
- 中文命名
- 文件名 = 一级标题

---

## 规则
- 必须更新 README.md
- 原则上只能改格式，不能改内容，除非有说明
- 更新项目规则精简
