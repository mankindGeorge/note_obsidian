# 笔记风格 Skill

## 笔记风格分析

通过分析现有笔记，总结出以下风格模式：

### 1. 标题层级

- `# 一级标题`：文件主题
- `## 二级标题`：主要章节
- `### 三级标题`：子章节或核心概念
- 无空行分隔标题

### 2. 代码块

- 标题与代码块之间空一行
- 代码块上方写说明文字
- 使用适当的语言标识符（如 `python`、`sql`、`bash`）
- 代码内中文注释

### 3. 列表

- **有序列表**：用于步骤（1. 2. 3.）
- **无序列表**：用于要点（- 或 *）
- 列表项使用 Tab 缩进
- 列表项之间不空行

### 4. 强调

- **加粗**：重点、概念、定义
- `行内代码`：参数、文件、路径、代码

### 5. 分隔

- 使用 `---` 分隔主要章节
- `---` 前后各空一行

### 6. 表格

- 用于对比、归纳
- 表头加粗

### 7. 图片

- 使用 `![](路径)` 语法
- 图片放在 `assets/文件名/` 目录

### 8. 引用

- 使用 `>` 标记补充说明、注意事项

## 创建笔记流程

### 1. 确定主题

- 文件名 = 一级标题（中文命名）
- 放在对应的分类目录下

### 2. 规划结构

```
# 笔记主题

## 第一章节

内容...

---

## 第二章节

### 子节

- 要点1
- 要点2

```代码语言
代码示例
```

> 补充说明
```

### 3. 编写内容

- 先写概念定义
- 再写代码示例
- 最后总结对比
